[![Open your Home Assistant instance and show the add-on store.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FNwus9XWK5UTy%2Fhomeassistant)

This repository hosts custom add-ons for Home Assistant.

## Available Add-ons

| Add-on | Description |
| :--- | :--- |
| **[Squid Proxy](squid-proxy/README.md)** | High-performance Squid proxy with Stealth Mode, HTTPS support, and monitoring. |

## Installation

1. Go to **Settings > Add-ons > Add-on Store** in Home Assistant.
2. Click the three dots (top-right) > **Repositories**.
3. Add the URL of this repository: `https://github.com/Nwus9XWK5UTy/homeassistant`.
4. The add-ons will appear in your store.
